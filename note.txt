////Git과Github에 대하여
서버(server) 컴퓨터 -> github(깃허브) -> jfs라는폴더를 생성함 -> readme.md파일이 자동생성됨
git(github를컨트롤하는것:자료up & down)과 github(실제 저장공간(repository))는 서로 다름
github- 파일을 담을수 있는 저장공간을 확보함
github=https://github.com/
standalone->설치후사용
portable->설치안함
repository가 바뀌면(추가되면) init부터 다시
장소가 변경되면 
-git config --global user.name "git에내이름"
-git config --global user.email "git에내이메일주소"
부터 다시한다.
후에 github에 올려져있는것 카피
-git clone https://github.com/내깃/jfs라는이름
만약 내컴에서 수정하면
-git add .
-git commit -m "message"
-git push -u origin main
으로 업로드 완료!
혹시 서버에 변경사항을 가져오려면
-git pull origin main
